This guide will help you troubleshoot and fix any connection issues with Pleasant Cove Design widgets.

Common Issues & Solutions

Messaging Widget Not Showing in Admin Inbox

This happens when your Admin UI is connected to a different backend than your widgets.

  • The Admin UI should be connected to: https://pcd-production-clean-production-e6f3.up.railway.app
  • Visit http://localhost:5173/set-api-prod.html to fix Admin UI connection

Widgets Not Working on Squarespace

Ensure proper integration by including the fix-widget-connections.js script in your widget code:

<script src="https://pcd-production-clean-production-e6f3.up.railway.app/client-widget/js/fix-widget-connections.js"></script>

Appointments Not Showing in Admin UI

The appointment widget must be configured to use the production API endpoint:

backendUrl = 'https://pcd-production-clean-production-e6f3.up.railway.app';
